# 🐾 MofuTube (モフチューブ) 運営・開発マニュアル

MofuTubeは、登録者数10万人以下の魅力的なYouTuber（動物、イケメン、可愛い、食べ物系）を美しく紹介するWebサイトと、それらをYouTube APIから自動収集して重複を排除するシステムが一体化したプロジェクトです。

開発環境（Node.jsやnpmなど）を必要とせず、ブラウザを開くだけで動作・収集・更新ができる超軽量設計になっています。

---

## 🚀 1. サイトの起動方法

1. プロジェクトフォルダ (`mofutube-project`) を開きます。
2. ルートディレクトリにある **`index.html`** をダブルクリックしてブラウザで開きます。
3. これだけで、サイトの閲覧、お気に入り機能、動画再生、および収集機能のすべてが動作します。
   * ※特別なサーバーやインストールの作業は一切不要です！

---

## 🛠️ 2. YouTuber自動収集ツールの使い方

画面右上の **「チャンネルを収集する」** ボタンから自動収集パネルを開きます。

1. **APIキー**: 後述の手順で取得した「YouTube Data APIキー」を入力します（キーはブラウザに安全に保存され、次回から入力不要になります）。
2. **キーワード**: 探したいチャンネルのキーワードを入力します。（例: `猫 癒やし`, `メンズメイク`, `大食い`, `可愛いDIY`）
3. **カテゴリー**: 追加したいカテゴリーを選択します。
4. **自動収集を開始する**: 
   * ボタンを押すと、条件に合う「登録者1,000人〜10万人以下」のYouTuberをYouTubeから自動で探し出します。
   * **重複防止機能**: すでに紹介済みのチャンネルは自動で判定され、除外（スキップ）されます。
5. **データの更新（永続化）**:
   * 収集したデータは一時的にブラウザ（localStorage）に保存され、サイトに即座に反映されます。
   * これを恒久的にサイトに登録するには、**「データをエクスポート」** ボタンを押して `channels.js` ファイルをダウンロードします。
   * ダウンロードした `channels.js` を、プロジェクト内の **`src/data/channels.js`** に上書き保存（ドラッグ＆ドロップで置換）してください。これで次回から最初から登録された状態になります。

---

## 🔑 3. YouTube Data APIキーの取得手順（無料・5分）

自動収集を動かすためのAPIキーは、以下の手順で無料で取得できます。

1. [Google Cloud Console](https://console.cloud.google.com/) にGoogleアカウントでログインします。
2. 画面上部のプロジェクト選択メニューから **「新しいプロジェクト」** を作成します（プロジェクト名は「MofuTube」など任意）。
3. 作成したプロジェクトを選択した状態で、メニューから **「APIとサービス」 > 「ライブラリ」** を開きます。
4. 検索窓に **「YouTube Data API v3」** と入力し、表示されたAPIを選択して **「有効にする」** ボタンを押します。
5. 有効化後、左メニューの **「認証情報」** を開き、画面上部の **「+ 認証情報を作成」 > 「APIキー」** を選択します。
6. 画面に表示された `AIzaSy...` から始まる長い文字列があなた専用の「APIキー」です。これをコピーして、MofuTubeの収集パネルに入力してください。

## 🎬 4. 顔出し・声出しなしの紹介動画（ショート）作成ガイド

YouTubeショートやTikTokに投稿する紹介動画を、顔出し・声出しなしで効率的に作るためのノウハウです。

### 📌 おすすめの制作手順
1. **台本を準備する**:
   * 私に「〇〇チャンネルを紹介する30秒のショート台本を作って」と指示してください。秒数に合わせた原稿を即座に作成します。
2. **AI音声を生成する**:
   * 無料の音声合成ソフト **「VOICEVOX（ボイスボックス）」** をパソコンにインストールします（簡単に入手できます）。
   * 私が作った台本のテキストを貼り付け、お好みのキャラクター（ずんだもん等）や、イケメン風のボイスで音声を書き出します（MP3形式）。
3. **画像・動画素材を用意する**:
   * 許可を得たYouTuberのチャンネルから、可愛いシーンのスクリーンショットや、数秒の切り抜き映像をダウンロードします。
4. **動画編集（コピペ＆ドラッグするだけ）**:
   * スマホやPCの無料編集アプリ（**CapCut** や **Vrew** など）を開きます。
   * 背景に動画素材を置き、VOICEVOXの音声を乗せます。
   * 音声に合わせて自動で字幕（テロップ）を生成する機能（CapCutやVrewに内蔵されています）を使い、文字をおしゃれに着色します。
5. **最後の誘導（CTA）**:
   * 動画の最後に「もっと見たい方はプロフィールリンクの『MofuTube』へ！」というスライドやテロップを入れて完了です。
